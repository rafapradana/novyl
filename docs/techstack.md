# Techstack Novyl AI

- Framework: Next.js (TypeScript + Tailwind CSS)
- UI Components: shadcn/ui
- ORM: Drizzle
- Auth Library: Better Auth
- Icons: Hugeicons
- UI Animation: Framer Motion
- Email: Resend
- AI Provider: OpenRouter
- AI Model: x-ai/grok-4.1-fast
- Durable Background Processes: Vercel Workflow SDK
- Database: Supabase (menggunakan Drizzle ORM dengan connection string di environment variables, bukan Supabase JS/TS SDK)
- Deployment: Vercel
